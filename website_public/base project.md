# Общий план разработки сайта amediå

Создание премиального одностраничного сайта для маркетингового агентства **amediå** с упором на сложную скролл-анимацию и скандинавский минимализм.

## Технологический стек
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS 4
- **Animations:** GSAP + ScrollTrigger (основной движок)
- **Smooth Scroll:** Lenis
- **Micro-interactions:** Framer Motion
- **Performance:** Canvas для Hero-анимации

## Дизайн-система
### Цвета
- **Dark Green (Primary):** `#274949` — основной глубокий фон.
- **Light Blue (Accent):** `#325EFC` — интерактивные элементы, кнопки, акценты.

## Структура сайта (One-page)

1.  **Hero Section:** 
    - Сцена на Canvas с последовательностью кадров (DJI Osmo Pocket 3).
    - Плавный переход от макро-фокуса на линзе к полному виду камеры.
    - Стадия 04: Камера как интерфейс с плавающими услугами.
2.  **About / Vision:**
    - Эффект "Focus to Vision" с reveal-анимацией текста.
3.  **ROI Calculator:**
    - Интерактивный расчет окупаемости с живым обновлением.
4.  **Portfolio (Case Studies):**
    - Горизонтальный скролл ленты проектов.
5.  **Lead Gen Form:**
    - Многошаговая форма-квалификатор.

## Этапы реализации

### Этап 1: Инфраструктура и Базовый Дизайн [DONE]

- Подключение Lenis.

### Этап 2: Hero-анимация (Core Feature) [DONE]
- Реализация Canvas-рендерера.

### Этап 3: Контентные секции [DONE]

### Этап 4: Интерактив и Формы [DONE]
- Создание многошаговой формы лидогенерации.

### Этап 5: Полировка и SEO [IN PROGRESS]
