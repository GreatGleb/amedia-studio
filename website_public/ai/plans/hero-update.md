# План: Обновление Главного Экрана (Hero Section)

Этот план описывает изменения на главном экране и реструктуризацию блока с анимацией.

## Цель
Сделать главный экран более привлекательным и информативным, а также улучшить подачу услуг.

## Предлагаемые изменения

### 1. Главный экран (Hero)
- **Фон:** Добавить живописную картинку (вместо текущего фона, если он был другим).
- **Контент:**
  - Яркий заголовок.
  - Краткий подзаголовок.
  - Кнопка действия (CTA).

### 2. Блок с анимацией секвенции (Sequence Animation)
- **Расположение:** Переместить блок ниже по странице.
- **Трансформация:**
  - Блок анимации будет располагаться **сбоку** (например, справа).
  - **Слева** от него будет список наших услуг.
- **Интерактивность:** При скролле/анимации каждая услуга должна увеличиваться и подсвечиваться.

## Стек и технологии
- **Анимации:** GSAP & Framer Motion.
- **Стили:** Tailwind CSS 4.
- **Плавный скролл:** Lenis.

## План действий
1. [ ] Провести ревью текущего кода главного экрана.
2. [ ] Подготовить ассеты (живописную картинку).
3. [ ] Реализовать новую структуру Hero секции.
4. [ ] Реализовать новую структуру блока услуг и секвенции.
5. [ ] Настроить анимацию увеличения и подсветки услуг с помощью GSAP.
