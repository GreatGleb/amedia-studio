# База знаний и инструменты ИИ

Эта папка содержит материалы для работы с ИИ, планы разработки и стандарты проектирования.

## Структура папок

- **/plans/** — Планы разработки (текущие, будущие, специфичные для фич).
- **/skills/** — Кастомные системные промпты и "скиллы" для ИИ (Antigravity, Roo Code).
- **/workflows/** — Описание процессов работы с ИИ и команд.
- **/patterns/** — Общие принципы и паттерны проектирования (DDD, KISS, SOLID, Best Practices).
- **/examples/** — Примеры качественного кода для переиспользования и обучения ИИ.

## Стек проекта
- **Framework:** Next.js
- **Runtime:** React 19
- **Styling:** Tailwind CSS 4
- **Animations:** GSAP & Framer Motion
- **Smooth Scroll:** Lenis
- **Language:** TypeScript
